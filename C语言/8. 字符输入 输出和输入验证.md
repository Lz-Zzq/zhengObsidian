#### 单字符I/O getchar与putchar
	 getchar与putchar每次只能处理一个字符 getchar与putchar并不是真正的函数，他们被定义为供预处理器使用的宏
#### 缓冲区
	缓冲分为两类：完全缓冲IO与行缓冲IO 
	完全缓冲：指的是缓冲区被填满时，刷新缓冲区。通常出现在文件输入中
	行缓冲：出现换行符时候刷新缓冲区。键盘输入通常是行缓冲
#### 文件结尾
	操作系统检测结尾方法之一是在末尾放一个特殊字符标记 ^z (Ctrl Z)
	或者文件有3000字节，那么就读取3000字节，报错Ctrl+Z

	无论操作系统选择哪种方式，getchar读到文件结尾会返回EOF(end of file)
	#define EOF (-1)    stdio.h文件中
	scanf()函数检测文件结尾也返回EOF
#### 重定向
```c
#include <stdio.h>
int	main() {
	int ch;
	while ((ch = getchar()) != EOF) {
		putchar(ch);
	}
	return 0;
}
```
	 echo_eof.exe < words.txt   在cmd中将words.txt文件内容重定向输入至exe文件中
	 echo_eof.exe > words.txt   在cmd中将echo_eof.exe执行，输入内容至words.txt的文件中
	 echo_eof.exe > words.txt < words2.txt 重定向组合，顺序无所谓但是不能重名，输入/输出
