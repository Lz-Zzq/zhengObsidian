#### sizeof运算符和size_t类型
>sizeof返回值是size_t类型的。
>C有一个typedef机制，允许自己创建类型别名。
```c
typedef double real;
//现在可以使用real来声明double类型变量了
real d;
```
#### 完整表达式
>C中序列点是程序执行的点，在该点上，所有的副作用都在进入下一步之前发生。C中一个；就是一个序列点，意思是<span style="color:yellow"> 在一个语句中，赋值运算符递增递减对运算对象做的改变<b>必须在程序执行下一条语句之前完成</b></span>
```c
while(guests++<10){     // 这一行是一个完整的表达式，所以递增会在该行结束进行
	printf("%d \n",guests);
}
```
