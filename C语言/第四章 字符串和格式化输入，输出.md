#### char类型数组和null字符
C语言没有专门存储字符串变量类型，字符串都被存储在char数组中。数组由连续的存储单元组成。
数组末尾位置 \0 是空字符，40存储单元字符串实际上只能存储39字符。
```c
scanf("%s",name)
> abc def
输出 abc
```
scanf（）一般只会读取第一个单词，它遇到空格，制表符换行符就不再读取输入

#### strlen（）函数
sizeof字节单位给出对象大小。strlen()函数给出字符串的字符长度。

#### 明示常量
使用C预处理器定义常量，
```c
# define TAXRATE 3.14
```
在编译程序时候所有TAXRATE都会替换成3.14 这一过程被称之为**编译时替换**

#### float转换说明
浮点类型，有double和long double类型转换，但是没有float的，因为会自动转换。printf()中所有float类型参数都会自动转换成double类型

#### 类型可移植性
sizeof运算符标准定义是无符号，实际上可以是unsigned int ... 因此，入股哦使用printf函数显示sizeof表达式，根据不同系统还需要切换转换说明。C提供了可移植性更好的类型。stddef.h头文件（包含在stdio.h）吧size_t定义成系统使用sizeof返回的类型，这被称之为底层类型(underlying type).其次，printf使用z修饰符打印相应类型。同样C还定义了ptrdiff_t类型和t修饰符来表示系统使用的两个地址差值的底层有符号整数类型。
