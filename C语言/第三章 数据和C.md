#### 位，字节，字
最小的存储单元是位（bit），可以存储0or1
字节（byte）是常用计算机存储单位。1字节8位。2的8次方256   0~255
字（word）是涉及计算机时给定的自然存储单位。8位，16位，32，64位，字长越大，数字转移速度越快。

#### 显示八进制十六进制
十进制 %d
八进制 %o
十六进制 %x
如果需要增加进制前缀 %#o 中间需要增加#

#### 整数类型
short int 简写short。占用存储空间可能比int小，有符号类型。
long int   long占用存储空间可能比int多，有符号类型
long long int     long long占用可能比long多至少占用64位，有符号类型。
unsigned 无符号，适用于非负数场合
signed 有符号，强调该值是有符号类型
当声明一个较小的long类型时候，通常 long i = 12L;**使用L后缀。longlong使用LL;**

#### 打印显示各种类型
打印unsigned int类型的值，使用 **%u**
long使用 **%ld** 如果计算机中long与int同样大，使用%d就行
**%lx**表示打印十六进制long类型整数
对于short类型，可以使用**h**前缀，**%hd**表示十进制显示short类型整数。
**%ho**表示显示八进制short类型整数
**%lld** 与 **%llu** 表示打印longlong有符号无符号类型

一般来说，short类型%hd还是int类型%d打印类型都一样，因为会把short转换成为int。因为int是计算机处理整数类型最高效的类型。
h修饰符可以显示较大整数被截断成short类型的情况
