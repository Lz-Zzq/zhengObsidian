## 二、程序分析——写出输出

**程序 A**

    n = 5
    while n > 0:
        n = n - 1
        if n == 2:
            continue
        print(n, end=' ')
    else:
        print('Done')

**程序 B**

    s = 0
    for i in range(1, 6):
        if i == 3:
            break
        s += i
    else:
        s = 100
    print(s)

**程序 C**

    for i in range(3):
        for j in range(3):
            if i == j:

## 四、编程练习（分层递进，每道题都有限制）

### ▶ 基础热身

**【4-1】反向输出字符串**  
字符串 `s = "HELLO"`，使用 **`while` 循环** 从最后一个字符开始逐个打印，每个字符占一行。  
**禁止**：`for`、切片、`reversed()`。

**【4-2】跳过 3 的倍数求和**  
使用 **`for` 循环 + `continue`**，求 1 到 30 之间所有**不是 3 的倍数**的整数之和，并打印结果。  
**禁止**：`while`、`range` 的步长参数直接跳过。

**【4-3】打印矩形边框**  
输入整数 `w` 和 `h`（均 ≥2），输出一个由 `*` 组成的 `w × h` 空心矩形。例如 `w=5, h=4` 输出：

    *****
    *   *
    *   *
    *****

**限制**：只能用 **`for` 嵌套 + `if`**，禁止使用字符串乘法 `'*' * w`、`while`、`break`、`continue`。

### ▶ 中级运用（灵活组合多个语法）

**【4-4】数字各位平方和**  
输入一个正整数，用 **`while` 循环** 计算该数每一位数字的平方和。例如输入 `34` 输出 `25`（即 3²+4²），只打印结果即可。  
**禁止**：字符串转换、`for`、列表。

**【4-5】寻找满足条件的数**  
找出 1～100 之间所有满足以下条件的数：该数本身既不能被 3 整除，其各位数字之和却能正好被 3 整除。逐个打印这些数。  
**要求**：必须使用 **`for` 循环**，内部用算术方法提取各位。**禁止**：`while`、字符串。

**【4-6】猜数字极简版（while + else）**  
设定目标数 `target = 58`，用户最多猜 **5** 次。每次输入后：
- 猜中：打印“猜中”并结束程序。
- 未猜中：提示“大了”或“小了”。
5 次都错，打印“机会用尽”。  
**要求**：必须包含 **`while`、`break`、`else`**，禁止使用 `for`、`continue`。

**【4-7】质数区间统计**  
输出 2～50 之间所有的质数，并**统计个数**，最后打印“共有 x 个质数”。  
**要求**：判断质数时必须用 **`for...else`**（即循环正常结束未触发 `break` 才判定为质数）。  
**禁止**：`while`。

### ▶ 进阶挑战（复杂但不极端，考察灵活应变）

**【4-8】打印空心菱形**  
输入一个**奇数** `n`（≥3），输出由 `*` 组成的空心菱形。例如 `n=5` 输出：

      *
     * *
    *   *
     * *
      *

**限制**：只能用 **`for` 嵌套 + `if`** 控制空格和星号的位置。  
**禁止**：字符串乘法 `*`、`while`、`break`、`continue`。

**【4-9】数字三角金字塔**  
输入 `n`（如 `n=5`），输出：

    1
    12
    123
    1234
    12345

**限制**：只能用 **`for` 嵌套** 和 `print(end="")` 控制输出。  
**禁止**：`while`、字符串乘法、列表。

**【4-10】模拟取款机次数控制**  
预设余额 1000 元，正确密码 `"1234"`。  
- 用户首先输入密码，最多 **3** 次机会，密码正确才能进入下一步。
- 密码正确后，可反复输入取款金额：
  - 输入 0 退出。
  - 金额大于余额提示“余额不足”，金额 ≤ 余额则扣款并显示剩余余额。
  - 一旦余额 ≤ 0，强制退出。
- 密码 3 次均错误，打印“已冻结”并结束。  
**要求**：密码验证部分必须用 **`while...else`** 实现冻结提示；取款部分用另一个 `while`。  
**禁止**：`for`。取款循环退出时允许使用 `break`，但密码验证循环**不能**用 `break` 直接跳出失败逻辑，必须通过条件让循环自然结束，从而触发 `else`。

**【4-11】筛选“自守数”**  
自守数是指一个数的平方的末尾几位等于该数本身。如 `5²=25`，末尾是 5；`25²=625`，末尾是 25。找出 1～1000 之间所有自守数并打印。  
**限制**：只能用 **`for` 循环** 和算术运算（`**`、`//`、`%` 等），不能使用字符串处理。  
**禁止**：`while`。

**【4-12】回文数字三角形**  
输入 `n`（如 4），输出：

    1
    121
    12321
    1234321

每行前后不能有多余空格。  
**限制**：只能用 **`for` 嵌套**，每行可用两个内部循环（一个递增一个递减）实现。  
**禁止**：`while`、字符串乘法、`break`、`continue`。
